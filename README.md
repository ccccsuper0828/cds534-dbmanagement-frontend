CDS534 前端界面


快速开始

### 安装依赖
```bash
npm install
```

### 启动开发服务器
```bash
npm start
```

应用将在 http://localhost:3000 启动

开发指南

1. **修改界面** - 编辑 `src/App.tsx`
2. **添加API** - 在 `src/services/api.ts` 中添加新方法
3. **样式调整** - 修改 `src/App.css`
4. **添加组件** - 在 `src/` 目录下创建新组件
